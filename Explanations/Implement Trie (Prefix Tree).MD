### **Example Trie**

```python
{
    'a': {
        'p': {
            'p': {
                '*': True,  # Marks "app" as a complete word
                'l': {
                    'e': {
                        '*': True  # Marks "apple" as a complete word
                    }
                }
            }
        }
    }
}
```
